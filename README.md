De volledige website is opgebouwd aan de hand van 
de kennis die werd aangeleerd binnen het vak Web-Essentials. 
Enkel voor de ‘speciale’ sectie werd AI (ChatGPT) ingezet om een 
teller-effect te realiseren. Deze teller is opgebouwd met HTML en CSS, 
waarbij gebruik wordt gemaakt van CSS-animaties om het gedrag van een 
mechanische kilometerteller te simuleren. Elk cijfer beweegt afzonderlijk 
en met een vaste snelheid per stap, wat zorgt voor een vloeiende en 
consistente visualisatie van mijn gelopen kilometers.

Verder werden geen afbeeldingen van het internet gebruikt. 
Alle afbeeldingen zijn zelfgemaakt.
